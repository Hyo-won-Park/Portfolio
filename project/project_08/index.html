KNOM

  • 블록체인 기술 기반

    KNOM 프로젝트의 메인 기술인 블록체인은 중앙 시스템에 대항하는 분산형 네트워크로서
    원하는 누구나 쉽게 접근하며 이용 가능

  • BUYER와 SELLER간의 유통 시스템

    오픈 마켓(mall)으로 스마트 스토어의 개념을 적용한 buyer 와 seller 간의 접근 간극을
    최소화 하여 직접적인 유통 시스템을 구축

  • 오픈 쇼핑몰

    KNOM 프로젝트는 블록체인 기술을 활용하여 세상에 단 하나뿐인 오픈 쇼핑몰

  • 신뢰 거래의 유통 플랫폼

    유일한 상품 인증 기술 등 차별화된 기술과 신뢰 보증 기관의 연계로 상품 거래의 신뢰를 주고 유통 시장을 혁신

PROJECT

PROJECT

KNOM 프로토콜는 블록체인을 기반으로 입점주와 소비자를 연결해주는 플랫폼입니다.

KNOM 프로토콜 구성 핵심

KNOM 프로토콜은 디지털 에셋 모델링 언어 기반으로 최적화된 디지털 자산을 설계하고 조정 통제합니다.

계약 모델링 3rd 파티 클라이언트 동의계약을 바탕으로 제공합니다.

FLATFORM

FLATFORM

  • POINT 01

    유일 프로덕트(상품) 정품 및 컨텐츠 인증

    실물 상품 + 디지털 NFT 암호화폐→상품의 정품 이력과 유일 인증키
    상품의 가치 및 품질을 스마트 컨트랙트로 보장

  • POINT 02

    NFT 경매형 거래 시스템

    한 국가에 제한되지 않고, 온라인을 통해 전 세계적으로 거래 가능
    환전, 외환거래, 해외송금을 쉽게

  • POINT 03

    유일상품 공동 구매 및 판매

    일반인들이 소액으로도 희소가치가 있는 유일 상품을 소유
    소유권은 KNOM 프로토콜과 이더리움 블록체인 서버에 저장
    KNOM 토큰을 통해서 희소상품 구매에 참여 가능
    투표(VOTING)을 통해서 재판매 또는 경매 여부 결정

  • POINT 04

    KNOM 이중 암호화폐 월렛 서비스

    NFT 토큰 → 플랫폼의 생태계는 유일 상품을 증명
    유틸리티 토큰 → 상품을 구매, 경매 참여
